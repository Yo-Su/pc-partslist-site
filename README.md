# 📝このアプリについて
自作パソコンのパーツリストの作成＆リストの比較ができるサイトです
- ユーザーを登録してパーツリストを保存できます
- パーツリストを作成できます
- パーツリストにパソコンのパーツを登録できます
- 作成したパーツリストを並べて比較できます
<br>
